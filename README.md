# Steps to Test Code:
1. Clone the below github repository

https://github.com/simigate/blockChain-project_5

2. Create `.secret.json` with Metamask mnemonic and Infura projectId as given in example below

{
    "mnemonic": "Your Metamask mnemonic",
    "projectId": "Your Infura projectId"
}

3. Go to project folder and install all the required npm packages in `package.json`using the command below :

npm install

4. Compile and Migrate to Rinkeby using the below commands:

truffle compile
truffle migrate --reset --network rinkeby 

5. Tokens can be minted using [myetherwallet](https://www.myetherwallet.com/)
I followed the steps of video mentioned in Appendix 2

6. Minted tokens can be listed and sold in [opensea](https://testnets.opensea.io/)
I followed the steps of video mentioned in Appendix 3

Testing:
1. Steps to test the code functionality using truffle test scripts:
truffle develop
test

4.Steps for using Zokrates 
 a: Install Docker
 b: Go to project folder and run ZoKrates docker container with the command below:
   docker run -v <path to your project folder>\zokrates\code:/home/zokrates/code -ti zokrates/zokrates:0.4.6 /bin/bash
 c:Go to folder code/square with the command below:
   cd code/square
 d:Compile the program written in ZoKrates DSL with the command below:
   ~/zokrates compile -i square.code
 e:Generate the Trusted Setup with the command below:
   ~/zokrates setup
 f:Compute Witness(the project already contains proof for [3,9]) with the command below:
   ~/zokrates compute-witness -a 3 9
 g:Generate Proof with the command below:
   ~/zokrates generate-proof
 h:Export Verifier with the command below:
   ~/zokrates export-verifier
